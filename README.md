# 🧠 XGMobile AI Course

Repositório destinado aos projetos, exercícios e anotações do curso de Inteligência Artificial da XGMobile.

---

### 📚 Trilha de Aprendizado

O conteúdo deste repositório reflete os módulos abordados durante o treinamento:

* **Módulo 1: Conceitos Gerais de IA:** Introdução, histórico da área e fundamentos de versionamento de código com Git e GitHub.
* **Módulo 2: Introdução ao ML:** Paradigmas de aprendizado (Supervisionado, Não Supervisionado e por Reforço) e técnicas essenciais de pré-processamento de dados.
* **Módulo 3: Aprendizado Supervisionado:** Implementação de algoritmos de Regressão (Linear, Múltipla, Polinomial, Logística), KNN e Árvores de Decisão. Inclui a análise de métricas de avaliação de modelos, como Matriz de Confusão e Curva ROC.
* **Módulo 4: Aprendizado Não Supervisionado:** Exploração de dados sem rótulos utilizando técnicas de Clustering, com foco no algoritmo K-Means.
* **Módulo 5: Projeto Prático:** Construção de um pipeline completo de Machine Learning, desde a análise de dados até a tomada de decisão.
