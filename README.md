# 🧠 XGMobile AI Course

Repositório destinado aos projetos, exercícios e anotações do curso de Inteligência Artificial da XGMobile.

---

### 📚 Trilha de Aprendizado

[cite_start]O conteúdo deste repositório reflete os módulos abordados durante o treinamento[cite: 863, 864, 865, 866, 867, 868]:

* [cite_start]**Módulo 1: Conceitos Gerais de IA:** Introdução, histórico da área e fundamentos de versionamento de código com Git e GitHub[cite: 864, 880, 922].
* [cite_start]**Módulo 2: Introdução ao ML:** Paradigmas de aprendizado (Supervisionado, Não Supervisionado e por Reforço) e técnicas essenciais de pré-processamento de dados[cite: 865, 975, 1007].
* [cite_start]**Módulo 3: Aprendizado Supervisionado:** Implementação de algoritmos de Regressão (Linear, Múltipla, Polinomial, Logística), KNN e Árvores de Decisão[cite: 866, 1140, 1348, 1423]. [cite_start]Inclui a análise de métricas de avaliação de modelos, como Matriz de Confusão e Curva ROC[cite: 1382, 1509].
* [cite_start]**Módulo 4: Aprendizado Não Supervisionado:** Exploração de dados sem rótulos utilizando técnicas de Clustering, com foco no algoritmo K-Means[cite: 867, 1637, 1656].
* [cite_start]**Módulo 5: Projeto Prático:** Construção de um pipeline completo de Machine Learning, desde a análise de dados até a tomada de decisão[cite: 868, 1689, 1703].
